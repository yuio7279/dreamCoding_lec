자바스크립트의 좌표는 항상 좌상

Element.getBoundingRect() : element 오브젝트 좌표에 대한 다양한 값들이 저장 되어 있다



top : 상측에서부터 오브젝트 시작까지의 길이

left : 좌측에서 부터 오브젝트 시작까지의 길이

width : 오브젝트의 가로

height : 오브젝트의 세로

right : 좌측에서 부터 오브젝트 끝까지의 길이

bottom : 상측에서 부터 오브젝트 끝까지의 길이


event.client x,y : document 현재 화면에서 해당 오브젝트와의 x,y값을 의미

event.page x,y : 전체 페이지 화면에서 해당 오브젝트와의 x,y값을 의미 (스크롤이 길수록 값은 더 커지게 됨)


body style { overflow : hidden } : 컨텐츠가 window를 벗어나면 스크롤을 못하게 할 수 있다


window.scrollBy(x,y) : x,y좌표만큼 이동

window.scrollTo(x,y) : x,y좌표로 이동
